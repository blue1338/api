🛠️ 主流工具配置指南

Claude Code

基于 Anthropic 协议的 CLI 工具配置方式：

macOS / Linux: 
export ANTHROPIC_BASE_URL="https://sparkide.top"
export ANTHROPIC_API_KEY="sk-您的令牌"
Windows (PowerShell): 
$env:ANTHROPIC_BASE_URL="https://sparkide.top"
$env:ANTHROPIC_API_KEY="sk-您的令牌"

Codex CLI

基于 OpenAI 协议的 CLI 工具配置方式：

macOS / Linux: 
export OPENAI_BASE_URL="https://sparkide.top/v1"
export OPENAI_API_KEY="sk-您的令牌"
Windows (PowerShell): 
$env:OPENAI_BASE_URL="https://sparkide.top/v1"
$env:OPENAI_API_KEY="sk-您的令牌"

Cursor IDE

⚠️ 注意：接入第三方 API 需拥有 Cursor Pro 账号。
  1. 打开 Cursor Settings
  2. 在侧边栏点击 Models
  3. 向下滚动找到 API Keys 并展开
  4. 开启 OpenAI API Key 开关，填入令牌
  5. 开启 Override OpenAI Base URL，填入： 
    https://sparkide.top/v1
